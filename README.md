# 🌊 BlockWave

BlockWave is a minimal educational blockchain implementation written in Python.  
It demonstrates the core mechanics behind cryptocurrencies such as Bitcoin and Ethereum.

---

## ✨ Features

- Block structure with SHA-256 hashing
- Proof-of-Work mining
- Transaction handling
- Wallet generation
- Chain validation
- Genesis block creation

---

## 🧠 Educational Purpose

This project is designed to help developers understand:

- How blockchain links blocks using hashes
- How mining and nonce work
- How transaction pools operate
- Basic wallet generation
- Blockchain validation logic
